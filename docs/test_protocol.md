# Протокол тестирования (Задание 1)

## 1. Объект тестирования
Функция расчета статистики: `services.calculate_statistics_from_rows`.

## 2. Инструмент
Pytest (автоматизированный модульный тест).

## 3. Входные данные
Три заявки:
1) выполнена, start=2023-01-01, completion=2023-01-06 (5 дней)
2) выполнена, start=2023-01-10, completion=2023-01-11 (1 день)
3) не выполнена (не финальная)

## 4. Ожидаемый результат
- total_requests = 3
- completed_requests = 2
- average_repair_time_days = (5 + 1) / 2 = 3.0

## 5. Фактический результат
Фактический результат должен совпасть с ожидаемым при запуске:
```bash
pytest -q
```
